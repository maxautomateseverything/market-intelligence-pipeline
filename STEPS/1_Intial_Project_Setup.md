## Add .gitignore

Open .gitignore and paste:

```
# Python
__pycache__/
*.pyc
.ipynb_checkpoints/

# Environment
.env
.venv/
venv/

# Data
data/*.duckdb
data/raw/*
data/processed/*

# VS Code
.vscode/

# OS
.DS_Store
Thumbs.db
```

.gitignore tells Git which files to ignore so they do not get committed to GitHub.

Here we are protecting the repo from uploading things that are:

- Autogenerated Python junk [```_pycache_/```, ```*.pyc```]
- Private secret keys/API keys [```.env```]
- Huge local environment folders [```venv/```, ```.venv/```]
- Raw data files that can be large or messy [```data/raw/*```, ```data/procesed/*```]
- The local database file [```data/*.duckdb```]
- Personal editor settings [```.vscode/```]
- Windows or Mac system files [```Thumbs.db```, ```.DB_Store```]

Therefore when we run ```git add . ``` Git will skil anything listed in ```.gitignore```.
