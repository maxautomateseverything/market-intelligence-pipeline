B. Installing Miniconda

Installed Miniconda from this website: https://www.anaconda.com/download/success

MiniConda was chosen as Anaconda Distribution contains all packages already installed
which we wont use. Miniconda gives us mroe control allowing us to choose what packages
are being installed.

After installing the download package we clicked through leaving all settings on the 
installer as defualt.

After installing successfully we open Anaconda Prompt. Anaconda Prompt runs Bash.
We then run ```conda --version``` and for this instance we can exepct a result 
```conda 25.x.x```
